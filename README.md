## Algoritmos y Estructuras de Datos

### ¿Qué es una estructura de datos?
Una Estructura de Datos es una ubicación nombrada que puede ser usada para almacenar y organizar datos.

Ejemplo: Un **Array** es una colección de elementos almacenados en ubicaciones de memoria contiguas.

``` javascript
['H', 'e', 'l', 'l', 'o'];
```

Diferentes Estructuras de Datos guardan y organizan datos de diferentes maneras, cada una tiene sus ventajas y desventajas.


### ¿Qué es un algoritmo?
Un algoritmo es un conjunto de pasos ordenados para resolver un problema.

Ejemplo: Búsqueda Lineal
<br/>
Compara los elementos uno a uno de un array hasta encontrar el valor buscado.
